Exemplo 1- Estudos de cores no CSS

As cores no CSS podem ser ultilizadas de varias maneiras,como por exemplo:

  • As cores podem ser representadas pelo nome, exemplo: red, green, blue, ...
  • As cores podem ser representadas através de uma função RGB, que é o vermelho, verde, e o azul
  • As cores, tambem podem ser representadas através de um número em hexadecimal
